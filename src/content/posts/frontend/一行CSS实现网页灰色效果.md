:::tip

今天发现很多网站，app,都变成了黑白色。通过了解发下安很多网站都只用了一个CSS属性便实现了。

这个属性似乎还有很多有意思的应用，不知道兼容性如何，之后有时间再研究一下

:::



```css
body {
    filter: grayscale(1);
}
```
